# Portfolio OS

A responsive desktop simulation portfolio built with semantic HTML, modern CSS layers, and vanilla DOM manipulation scripts. This project showcases a Windows-inspired interface with interactive windows, authentication, and a modern glassmorphism design.

## Features

- 🔐 **Authentication System** - Command prompt-style login with admin credentials
- 🖥️ **Desktop Interface** - Windows-inspired desktop with draggable windows
- 📁 **Interactive Windows** - About me and Projects sections with smooth animations
- ⏰ **System Clock** - Real-time taskbar clock showing date and time
- 🎨 **Modern Design** - Glassmorphism effects with backdrop blur
- 🎯 **Drag & Drop** - Fully draggable window system with z-index management
- 📱 **Responsive** - Works seamlessly across different screen sizes

## Project Structure

```
Portfolio-OS/
├── index.html      # Main HTML structure
├── style.css       # Styling with CSS layers and animations
├── script.js       # DOM manipulation and interactivity
└── README.md       # Project documentation
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with backdrop filters and animations
- **Vanilla JavaScript** - DOM manipulation and event handling
- **Iconify** - SVG icon library integration

## Key Components

### Authentication System
Command prompt-style login interface that mimics Windows authentication.

### Window Management
- Draggable windows with realistic z-index layering
- Smooth open/close animations
- Window controls (close button)

### System Clock
Real-time clock in the taskbar showing current date and time.

### Start Menu
Windows 7-styled start menu with quick access to applications.

## Customization

You can easily customize this portfolio by:

- Editing the About section in the `about-win` window
- Adding new projects to the `projects-win` window
- Modifying colors in CSS variables (`:root`)
- Adding new windows and desktop shortcuts
- Changing the authentication credentials in `script.js`
